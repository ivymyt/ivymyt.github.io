Text Countlet

Text Countlet is a Javascript bookmarklet that allows you to quickly count characters, words, and paragraphs on a webpage. It takes care of end-of-line hyphenation and word-separating punctuation (e.g. ellipses and dashes).

Last updated in 2010, so it likely does not work in modern browsers.

Installation

Bookmark Text Countlet

Drag it into your Bookmarks Bar or Right-click and select “Bookmark This Link” or “Add to Favourites”.

Modes

  • Character count
  • Word count
    • Ignore punctuation — All punctuation marks are treated as letters.
  • Paragraph count
    • 1 line break ⇒ a new paragraph
    • 2 line breaks ⇒ a new paragraph
